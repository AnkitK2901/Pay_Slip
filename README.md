# Pay_Slip

This Java-based application generates pay slips for employees, supporting both permanent and contractor staff. It calculates salaries based on employee type and displays the generated pay slips.

## Features

- **Employee Types**: Handles both permanent and contractor employees.
- **Salary Calculation**: Computes salaries based on employee classification.
- **Pay Slip Generation**: Produces pay slips for each employee.
